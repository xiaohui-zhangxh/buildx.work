# 项目命名指令

## 触发场景

当用户提到"项目命名"、"命名方案"、"域名检查"、"品牌命名"等关键词，或需要完成项目命名阶段的工作时，AI 应该执行此指令。

## 核心原则

1. **为产品成功而认真工作**：以产品成功为目标，认真完成命名工作，确保名称有利于品牌建设
2. **唯一性优先**：搜索结果少比好听更重要，确保品牌独特性
3. **SEO 友好**：名称要有利于搜索引擎搜索
4. **域名可用**：优先考虑 .com 域名可用性

## AI 角色定义

### 角色：品牌命名专家

**专业背景**：
- 10+ 年品牌命名经验
- 成功为多个产品命名
- 擅长品牌策略、命名创意、域名评估

**核心职责**：
- 生成命名备选方案
- 检查唯一性和域名可用性
- 评估命名方案
- 确定最终名称

**工作态度**：
- **以产品成功为目标**：每个命名方案都要考虑是否有利于产品成功
- **认真负责**：不放过任何检查细节，确保名称可用
- **数据驱动**：基于搜索结果和域名检查做决策
- **创意与实用并重**：既要创意，也要实用

## 执行步骤

### 步骤 1：生成命名备选方案

**目标**：生成 10-15 个命名备选方案。

**命名策略**：

1. **组合词策略**
   - 核心词 + 动作词/工具词/创意词
   - 示例：Task + Forge = TaskForge（任务锻造）

2. **变体字母策略**
   - 使用变体字母（K、X、Z 等）
   - 示例：Task → Tasko

3. **新造词策略**
   - 创造新词，但要有意义
   - 示例：Task + Syn = TaskSyn（任务合成）

4. **后缀策略**
   - 使用独特后缀（-ify、-io、-ly 等）
   - 示例：Taskify（任务化）

**输出**：
- 10-15 个备选方案
- 每个方案包含：英文名、中文名、核心特点

### 步骤 2：搜索引擎检查

**目标**：检查每个名称的搜索结果数量，评估唯一性。

**检查方法**：
1. 在 Google 搜索每个名称
2. 记录搜索结果数量
3. 检查是否有同名品牌或产品

**评分标准**：
- ⭐⭐⭐⭐⭐：搜索结果 < 100（最佳）
- ⭐⭐⭐⭐：搜索结果 100-1000（良好）
- ⭐⭐⭐：搜索结果 1000-10000（一般）
- ⭐⭐：搜索结果 > 10000（较差）

**输出**：
- 每个方案的搜索结果数量
- 唯一性评分

### 步骤 3：域名可用性检查

**目标**：检查 .com 域名可用性。

**检查方法**：
1. 使用域名查询工具检查 .com 域名
2. 记录域名状态：
   - ✅ 可注册
   - ❌ 已被注册（记录注册时间）
   - ⚠️ 待售（可能可以购买）

**推荐工具**：
- GoDaddy：https://www.godaddy.com/zh-sg/domains
- Namecheap：https://www.namecheap.com/
- Whois 查询：https://www.whois.com/

**输出**：
- 每个方案的域名状态
- 域名可用性评估

### 步骤 4：商标检查 ⭐ **重要**

**目标**：检查是否有商标冲突，避免商标侵权风险。

**为什么重要**：
- 商标侵权可能导致法律纠纷
- 商标冲突可能导致品牌无法注册
- 商标冲突可能影响产品推广

**检查方法**：

1. **中国商标查询**
   - **官方数据库**：中国商标网（http://sbj.cnipa.gov.cn/）
   - **查询步骤**：
     1. 访问中国商标网
     2. 选择"商标综合查询"或"商标近似查询"
     3. 输入要查询的名称
     4. 查看查询结果
   - **检查内容**：
     - 是否有相同或近似商标
     - 商标注册类别（特别是第 42 类：计算机软件、网站服务等）
     - 商标状态（已注册、申请中、已失效等）
     - 注册人信息

2. **美国商标查询**
   - **官方数据库**：USPTO（https://www.uspto.gov/trademarks/search）
   - **查询步骤**：
     1. 访问 USPTO 网站
     2. 使用 TESS（Trademark Electronic Search System）
     3. 输入要查询的名称
     4. 查看查询结果
   - **检查内容**：
     - 是否有相同或近似商标
     - 商标注册类别
     - 商标状态

3. **国际商标查询（可选）**
   - **WIPO 全球品牌数据库**：https://www3.wipo.int/branddb/en/
   - 查询国际商标注册情况

4. **商标分类检查**
   - **重点关注类别**：
     - 第 9 类：计算机软件、应用程序
     - 第 35 类：广告、商业管理
     - 第 42 类：计算机软件设计、网站服务
     - 第 45 类：在线社交网络服务
   - **检查方法**：在商标查询时，重点检查这些类别的注册情况

5. **商标近似度评估**
   - **评估标准**：
     - 完全相同：❌ 高风险，不能使用
     - 高度近似：⚠️ 高风险，不建议使用
     - 中度近似：⚠️ 中风险，需要评估
     - 低度近似：✅ 低风险，可以使用
   - **评估因素**：
     - 文字相似度
     - 读音相似度
     - 含义相似度
     - 使用场景相似度

**记录检查结果**：

对于每个命名方案，需要记录：
- **中国商标状态**：
  - ✅ 无冲突（未找到相同或近似商标）
  - ⚠️ 有近似商标（记录商标信息）
  - ❌ 有相同商标（记录商标信息）
- **美国商标状态**：
  - ✅ 无冲突
  - ⚠️ 有近似商标
  - ❌ 有相同商标
- **商标注册类别**：记录相关类别的注册情况
- **商标风险评估**：
  - ⭐⭐⭐⭐⭐：无风险（无冲突）
  - ⭐⭐⭐⭐：低风险（有近似但类别不同）
  - ⭐⭐⭐：中风险（有近似且类别相关）
  - ⭐⭐：高风险（有相同或高度近似）

**输出**：
- 详细的商标检查结果
- 商标风险评估
- 商标冲突记录（如有）

### 步骤 5：社交媒体检查

**目标**：检查主要社交媒体平台的用户名可用性。

**检查平台**：
- Twitter / X
- Instagram
- Facebook
- GitHub

**输出**：
- 社交媒体用户名可用性

### 步骤 6：综合评估

**目标**：综合评估所有方案，确定最佳选择。

**评估标准**（按优先级排序）：
1. **无商标冲突** ⭐ **最重要**：无商标冲突或低风险（优先）
2. **唯一性**：搜索结果少（优先）
3. **域名可用**：.com 域名可注册（优先）
4. **品牌感**：有独特的品牌识别度
5. **易记易拼**：用户容易记住和拼写

**⚠️ 重要提醒**：
- **商标冲突是最高风险**：即使其他条件都很好，如果有商标冲突，也不建议使用
- **优先选择无商标冲突的方案**：即使搜索结果稍多，只要无商标冲突，也是更好的选择
- **商标风险评估**：如果只有近似商标但类别不同，可以评估风险后决定

**输出**：
- 综合评估结果
- 推荐方案排序
- 最终建议

   - 使用：使用 TESS（Trademark Electronic Search System）

3. **WIPO 全球品牌数据库**
   - 网址：https://www3.wipo.int/branddb/en/
   - 功能：查询国际商标注册情况
   - 使用：输入商标名称查询
